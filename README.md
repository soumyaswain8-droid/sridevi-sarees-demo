# StockPilot — Sri Devi Sarees (Interactive Demo)

Live demo of **StockPilot by Surya AI** — saree inventory, billing & loss-prevention for textile retail.

**Open the demo:** [index.html](./index.html) · **Live:** https://soumyaswain8-droid.github.io/sridevi-sarees-demo/

The full app runs client-side (no install, no login). Try:
- **Dashboard** — click any KPI card, donut segment, bar or chart to drill in
- **Inventory** — filter Fast / Slow / Dead; click a row for saree detail
- **Intake (Truck)** — generate SKUs and print barcode tags
- **Billing (POS)** — add a saree, then "Simulate swapped tag" to see the fraud-catch
- **Reports** — capital, ageing, margin by category

All data is fictional ("Sri Devi Sarees"). Pitch deck: [StockPilot-Deck.pdf](./StockPilot-Deck.pdf).
